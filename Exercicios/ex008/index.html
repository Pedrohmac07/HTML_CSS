Tipos de formatações

Negrito / Destacar

Para utilizar Negrito só precisa utilizar a tag:b (não é semantica)

para destacar uma palavra usando tag:strong (semantica)

Italico / Ênfase

Para utilizar o italico é preciso utilizar a tag:i (não é semantica)

Para dar enfase a uma palavra utiliza a tag:em (semantica)

Texto Marcado

Podemos criar um texto marcado usando a tag:mark

outro paragrafo outro texto marcado! (semantica)

Texto pequeno / Texto grande

Para criar um texto pequeno usamos a tag:small (semantica)

Para criar um texto grande usamos a tag:big (não é semantica)

Inserido/deletado

Para criarmos textos inseridos sublinhados por baixo utilizamos a tag:ins

Para criarmos texto riscado como se fosse desconsiderado, utilizamos a tag:del

Texto sobrescrito / texto subscrito

Para criarmos um texto sobrescrito, por exemplo x2 utilizamos a tag:sup

Para criarmos um texto subscrito, por exemplo h2o utilizamos a tag:sub

Por que deveriamos utilizar as semanticas?

As tags não semanticas, são antigas e descontinuadas por causa da integração com o CSS, por isso é recomendado utilizar as semanticas!
